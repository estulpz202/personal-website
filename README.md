## Getting Started

To run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Formatting & Linting

Check and fix style issues:

```bash
npm run lint
npm run format
```

## Deployment
